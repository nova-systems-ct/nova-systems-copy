# Nova Systems

Marketing and product website for Nova Systems — premium web infrastructure, AI-driven lead capture, and presence kiosks.

## Stack

- React 18 + Vite
- Tailwind CSS v3
- React Router v6
- Radix UI / shadcn components
- Tanstack Query

## Local development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy

Push to `main` — Vercel deploys automatically.
